The Features of Your New Website

Interactive

Your website will be interactive using JavaScript code to allow the site to react to the users’ actions, thus expanding its functionality.

Responsive

Your website will be responsive to the dimensions of its users’ viewports so that it looks good on desktop and laptops screens as well as tablets and mobile phones.

Backend/Database*

Your website may have a backend or database element to run server-side. This will allow it to store information like products, orders, and more.

*Sites with a backend may be more difficult and more expensive to host, and they will likely be priced higher.

Developer Experience

An experienced developer is crucial for making professional websites. Be assured, however, as your developer will have experience in the following web-development languages, libraries, and frameworks.

HTML

HTML or HyperText Markup Language is the core of web development. Responsible for the content of a site, a good knowledge of HTML is crucial to effectively building websites.

CSS

CSS or Cascading Style Sheets is used for stylizing a site. It is important to allow a site to have a "look" to it rather than looking like a plain text document.

JavaScript

JavaScript is at the core of web interactivity. It is used to allow users to interact with your website beyond the basic interactivity provided by HTML & CSS.

jQuery

jQuery is a lightweight JavaScript library that makes manipulating the Document Object Model (or DOM) easier, allowing for more development with less code.

ReactJS

ReactJS or React is an extension of JavaScript that allows for JSX or JavaScript XML files that have additional functionality and can make it easier to quickly build web apps.

NodeJS

NodeJS or Node is a JavaScript runtime environment that allows JavaScript to be run on a server and thus converts it into a backend server-side programming language.
